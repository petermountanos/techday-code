Class graphicsTriangle¶

class breakoutGraphics.graphicsTriangle(**keywords)[source]¶

Instances represents a triangle in the GameView

The triangle is defined by the attribute points which is a list of 6 floats. It stores the three x-y pairs that define the triangle. The object uses fillColor for the interior of the triangle and lineColor for its borders.

This class is a subclass of graphicsObject, and inherits all of its attributes and methods.

Attributes¶

These attributes are in addition to those from graphicsObject.

graphicsTriangle.points¶

Property that represents a list.

Only lists are allowed. Tuple or any other classes are forbidden.

graphicsTriangle.height¶

This attribute is computed automatically from the attribute points

graphicsTriangle.width¶

This attribute is computed automatically from the attribute points

graphicsTriangle.x¶

This attribute is computed automatically from the attribute points

graphicsTriangle.y¶

This attribute is computed automatically from the attribute points
