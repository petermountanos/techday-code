Class graphicsImage¶

class breakoutGraphics.graphicsImage(**keywords)[source]¶

Instance represents a rectangular image in the GameView

The image is given by a JPEG, PNG, or GIF file whose name is stored in the attribute imageSource. Image files should be stored in the Images directory, so that Kivy can find them without the complete path name. In this graphics object, the lineColor and fillColor attributes are ignored. The image is displayed as a recntangle whose bottom left corner is defined by attribute pos and whose width and height are defined by the atribute size. If the size attribute does not agree with the actual size of the image, the image is scaled to fit.

This class is a subclass of graphicsObject, and inherits all of its attributes and methods.

Attributes¶

These attributes are in addition to those from graphicsObject.

graphicsImage.source¶

File name of a source file for this image.

Invariant: Must be a string or None. Default is None.
graphicsImage.fillcolor¶

This attribute is ignored.

graphicsImage.linecolor¶

This attribute is ignored.
