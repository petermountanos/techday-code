Class graphicsLine¶

class breakoutGraphics.graphicsLine(**keywords)[source]¶

Instance represent a sequence of line segments in the GameView

The line is defined by the points attribute which is an (even) sequence of alternating x and y values. When added to the graphics view, the view draws a sequence of line segments, starting from one x-y pair in points and going to the next x-y pair. If points has a length 2n, then the result is n-1 line segments.

The object uses the attribute lineColor to determine the color of the line. The attribute fillColor is unused (even though it is inherited from graphicsObject).

This class is a subclass of graphicsObject, and inherits all of its attributes and methods.

Attributes¶

These attributes are in addition to those from graphicsObject.

graphicsLine.points¶

Property that represents a list.

Only lists are allowed. Tuple or any other classes are forbidden.

graphicsLine.fillcolor¶

This attribute is ignored.

graphicsLine.height¶

This attribute is computed automatically from the attribute points

graphicsLine.width¶

This attribute is computed automatically from the attribute points

graphicsLine.x¶

This attribute is computed automatically from the attribute points

graphicsLine.y¶

This attribute is computed automatically from the attribute points
